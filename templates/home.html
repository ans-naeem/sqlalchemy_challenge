These are the static route for my website

  • /api/v1.0/precipitation
    • it retrieves only the last 12 months of data to a dictionary using date as the key and prcp as the value.
  • /api/v1.0/stations
    • Return a JSON list of stations from the dataset.
  • /api/v1.0/tobs
    • it return the temperature observations of the most-active station for the previous year of data in json list format.

The Dynamic routes of my website look like this:

  • /api/v1.0/start_date
  • /api/v1.0/start_date/end_date
